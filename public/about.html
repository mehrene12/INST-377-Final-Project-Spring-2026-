About CareerPath

A career planning tool built for students by students.

📌 What is CareerPath?

CareerPath is a web application designed to help students explore career options, compare salaries, and research grad school opportunities around the world. Built as a final project for INST 377 at the University of Maryland.

🛠️ APIs Used

USAJobs API

Official US government job listings with real salary data from data.usajobs.gov

REST Countries API

Country information including population, currency, and location from restcountries.com

Supabase

Our own database for storing and retrieving saved career paths.

❓ How to Use

1. Go to the Jobs page to search US government job listings by keyword.
2. Save interesting careers using the ⭐ Save button.
3. Go to Grad School to explore countries for graduate study.
4. Return to Home to see your saved careers and salary comparisons.
